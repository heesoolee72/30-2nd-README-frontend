## 필수 구현 사항
1. 소셜로그인 (회원가입 / 로그인)
2. 메인페이지 
- 슬라이드, 전체 카테고리 보기(별도의 페이지로 분리하지 않음)
- 상품 리스트 - 체크박스, 필터링, 페이지네이션 (메인페이지와 합치는 것을 고려)
3. 도서 상세페이지 - 더보기(페이지네이션), 리뷰, 
4. 장바구니 - 체크박스
